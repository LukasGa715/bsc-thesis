# bsc-thesis
Stochastic Optimal Control for an Investment Portfolio: A Strategy with Simulations and Empirical Data
MATLAB code for the bachelor's thesis "Stochastic Optimal Control for an
Investment Portfolio: A Strategy with Simulations and Empirical Data"
(Mälardalen University, MAA043, 2026). Run naujausiaskodas.m with
stock_data.csv in the same folder. Requires the Econometrics Toolbox
(autocorr).
